Key Features

Multi-Model Support

Support for multiple model providers including OpenAI, Anthropic, Gemini, Azure OpenAI, and custom endpoints.

Cross-Platform

Works across Windows, macOS, and Linux with dedicated knowledge bases for each platform.

Web Knowledge Retrieval

Perplexica integration for retrieving information from the web to enhance agent capabilities.

Enhanced Memory

Episodic and narrative memory systems that enable learning from past experiences.

SOTA Performance

State-of-the-art results on benchmarks like OSWorld, WindowsAgentArena, and AndroidWorld.

Developer-Friendly

Easy-to-use SDK and CLI interface for integrating Agent S2 into your applications.
