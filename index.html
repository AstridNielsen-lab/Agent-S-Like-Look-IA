Abstract

We present Agent S2, a compositional generalist-specialist framework for computer use agents. Our approach combines a generalist LLM with specialized modules for enhanced perception, planning, and execution across operating systems. Agent S2 leverages multi-modal understanding, episodic memory, and procedural knowledge to achieve state-of-the-art performance on benchmark tasks.

Agent S2 demonstrates superior performance on OSWorld, WindowsAgentArena, and AndroidWorld benchmarks, outperforming previous approaches by significant margins. Our framework enables autonomous interaction with graphical user interfaces across multiple platforms without requiring specialized APIs or application-specific knowledge.

Method

Agent S2 uses a compositional architecture that integrates multiple specialized modules:

Perception

Multi-modal understanding of screen content and accessibility tree information to comprehend GUI state.

Planning

Hierarchical planning with task decomposition and directed acyclic graph (DAG) execution for complex multi-step tasks.

Memory

Episodic and narrative memory systems that enable learning from past experiences and applying knowledge to new scenarios.

Execution

Low-level action execution through our Agent-Computer Interface (ACI) that works across operating systems.

Architecture

Agent S2 integrates a generalist large language model with specialized modules for perception, planning, and execution. This compositional approach allows the agent to benefit from the general capabilities of LLMs while leveraging domain-specific knowledge for computer interaction tasks.

Key Features

Multi-Model Support

Supports OpenAI, Anthropic, Gemini, Azure OpenAI, and custom endpoints.

Cross-Platform

Works across Windows, macOS, and Linux with dedicated knowledge bases.

Web Knowledge Retrieval

Perplexica integration for retrieving web information to enhance capabilities.

Implementation and Usage

Installation

Install Agent S2 using pip:

pip install gui-agents

Basic Usage

After installation, you can run Agent S2 with the following Python code:

from gui_agents.s1.core.AgentS import GraphSearchAgent
from gui_agents.s1.aci.WindowsOSACI import WindowsACI

# Initialize the grounding agent for your OS
grounding_agent = WindowsACI()

# Configure the engine parameters
engine_params = {
    "engine_type": "openai",  # or "anthropic", "gemini"
    "model": "gpt-4o",
    "api_key": "your_api_key_here"
}

# Create the agent
agent = GraphSearchAgent(
    engine_params,
    grounding_agent,
    platform="windows",
    action_space="pyautogui",
    observation_type="mixed"
)

# Reset and run the agent with an instruction
agent.reset()
agent.run("Open Notepad and type 'Hello, world!'")
